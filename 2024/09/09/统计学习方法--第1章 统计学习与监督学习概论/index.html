统计学习方法--第1章 统计学习及监督学习概论

发表于2024-09-09
|阅读量:

1.统计学习简述

  统计学习是关于计算机基于数据构建概率统计模型，并运用模型对数据进行预测和分析的一门学科，又称统计机器学习。

1.1 统计学习特点

  学习：如果一个系统能够通过执行某个过程改进它的性能，这就是学习；
  统计学习：计算机系统通过运用数据、及其方法提高系统性能的机器学习。

  • 以计算机及网络为平台
  • 以数据为研究对象，对数据进行预测分析。基本假设：同类数据具有一定统计规律性。书中只讨论离散变量、不考虑数据观测与获取。
  • 以方法为中心，构建并应用模型
  • 交叉学科：概率论、统计学、信息论、计算理论、最优化理论

1.2 统计学习三要素

  • 从给定的、有限的的训练数据出发，假设数据是独立同分布的
  • 假设要学习的模型属于某个函数的集合，称为假设空间
  • 应用某个评价标准（策略），从假设空间选出最优模型，使得对已知训练数据、未知测试数据在评价标准下有最优预测
  • 算法实现

1.3 统计学习研究

  • 统计学习方法：开发新的方法
  • 统计学习理论：探究方法的有效性与效率
  • 统计学习应用：解决实际问题

2.统计学习分类

2.1 基本分类

  监督学习、无监督学习、半监督学习、强化学习

2.2 按模型分类

2.2.1 概率模型、非概率模型

  概率模型：决策树、朴素贝叶斯、隐马尔可夫模型、条件随机场、概率潜在语义分析、潜在狄利克雷分配、高斯混合模型、逻辑回归
  非概率模型：感知机、支持向量机、K近邻、AdaBoost、K均值、潜在语义分析、神经网络、逻辑回归

  • 监督学习中：
    • 概率模型取 P(y∣x)P(y|x)P(y∣x)，称为生成模型
    • 非概率模型取 y=f(x)y=f(x)y=f(x)，称为判别模型
  • 无监督模型中：
    • 概率模型取 P(z∣x)P(z|x)P(z∣x) 或 P(x∣z)P(x|z)P(x∣z)
    • 非概率模型取 z=g(x)z=g(x)z=g(x)
2.2.2 线性模型、非线性模型

  非概率模型中，又可分为：线性模型、非线性模型。

  • 判别准则：y=f(x)y=f(x)y=f(x)、z=g(x)z=g(x)z=g(x)是线性函数，则称为线性。

  线性模型：感知机、线性支持向量机、K近邻、K均值、潜在语义分析
  非线性模型：核函数支持向量机、AdaBoost、神经网络

2.2.2 参数化模型、非参数化模型

  参数化模型：模型参数的维度固定，模型由有限维参数完全刻画。感知机、朴素贝叶斯、逻辑回归、K均值、高斯混合模型。
  非参数化模型：参数维度不固定（无穷），随机数据量增加而增加。决策树、支持向量机、AdaBoost、K近邻、潜在语义分析、概率潜在语义分析、潜在狄利克雷分配。

2.3 按算法分类：在线学习与批量学习

  在线学习：一次接受一个样本，而后预测、学习模型，不断重复。
  批量学习：一次性接受所有数据，学习模型，而后预测。

2.4 按技巧分类：贝叶斯学习、核方法

3. 统计学习三要素：模型、策略、算法

3.1 模型是什么?

  在监督学习过程中，模型就是所要学习的条件概率分布或者决策函数。模型的假设空间 \cal F 包含所有可能的条件概率分布或决策函数。

假设空间 F\mathcal{F}F 输入空间 X\mathcal{X}X 输出空间 Y\mathcal{Y}Y 参数空间
决策函数 {f∣Y=fθ(x)}\{ f \mid Y = f_{\theta}(x)\}{f∣Y=fθ​(x)} 变量 变量 θ∈Rn\theta \in \mathbb{R}^nθ∈Rn
条件概率分布 {P∣Pθ(Y∣X)}\{ P \mid P_{\theta}(Y \mid X)\}{P∣Pθ​(Y∣X)} 随机变量 随机变量 θ∈Rn\theta \in \mathbb{R}^nθ∈Rn

3.2 策略是什么?

3.2.1 损失函数与风险函数

损失函数度量模型一次预测的好坏，风险函数度量平均意义下模型预测的好坏。

  1. 损失函数(loss function) 或 代价函数(cost function)：
    给定输入XXX的 预测值f(X)f(X)f(X) 和 真实值YYY 之间的非负实值函数，记作L(Y,f(X))L(Y,f(X))L(Y,f(X))

  2. 风险函数(risk function) 或 期望损失(expected loss)：
    与泛化误差的形式是一样的，

    Rexp(f)=Ep[L(Y,f(X))]=∫X×YL(y,f(x))P(x,y) dxdyR_{exp}(f)=E_p[L(Y, f(X))]=\int_{\mathcal X\times\mathcal Y}L(y,f(x))P(x,y)\, {\rm d}x{\rm d}y Rexp​(f)=Ep​[L(Y,f(X))]=∫X×Y​L(y,f(x))P(x,y)dxdy

    即，模型f(X)f(X)f(X)关于联合分布P(X,Y)P(X,Y)P(X,Y)的平均意义下的损失(期望损失)，反映了对整个数据的预测效果的好坏.

  3. 经验风险(empirical risk)或经验损失(empirical loss)：
    模型fff关于训练样本集的平均损失

    Remp(f)=1N∑i=1NL(yi,f(xi))R_{emp}(f)=\frac{1}{N}\sum^{N}_{i=1}L(y_i,f(x_i)) Remp​(f)=N1​i=1∑N​L(yi​,f(xi​))

    根据大数定律，当样本容量N趋于无穷大时，经验风险趋于期望风险。

  4. 结构风险(structural risk)：
    用以权衡经验风险和模型复杂度

    Rsrm(f)=1N∑i=1NL(yi,f(xi))+λJ(f)R_{srm}(f)=\frac{1}{N}\sum_{i=1}^{N}L(y_i,f(x_i))+\lambda J(f) Rsrm​(f)=N1​i=1∑N​L(yi​,f(xi​))+λJ(f)

    J(f)J(f)J(f)为模型复杂度, λ⩾0\lambda \geqslant 0λ⩾0是系数。

3.2.2 常用损失函数

  损失函数数值越小，模型就越好

  1. 0-1损失
    L(Y,f(X))={1,Y≠f(X)0,Y=f(X)L(Y,f(X))=\begin{cases}1, Y \neq f(X) \\0, Y=f(X) \end{cases}L(Y,f(X))={1,Y​=f(X)0,Y=f(X)​
  2. 平方损失
    L(Y,f(X))=(Y−f(X))2L(Y,f(X))=(Y-f(X))^2L(Y,f(X))=(Y−f(X))2
  3. 绝对损失
    L(Y,f(X))=∣Y−f(X)∣L(Y,f(X))=|Y-f(X)|L(Y,f(X))=∣Y−f(X)∣
  4. 对数损失
    L(Y,P(Y∣X))=−log⁡P(Y∣X)L(Y,P(Y|X))=-\log P(Y|X)L(Y,P(Y∣X))=−logP(Y∣X)
    前面负号：P(Y∣X)⩽1P(Y|X)\leqslant 1P(Y∣X)⩽1
3.2.3 ERM与SRM

  经验风险最小化(ERM)与结构风险最小化(SRM)

  1. 极大似然估计是经验风险最小化的一个例子
    当模型是条件概率分布，损失函数是对数损失函数时，经验风险最小化等价于极大似然估计
  2. 贝叶斯估计中的最大后验概率估计是结构风险最小化的一个例子
    当模型是条件概率分布，损失函数是对数损失函数，模型复杂度由模型的先验概率表示时，结构风险最小化等价于最大后验概率估计

3.3 算法

4. 模型评估与模型选择

  训练误差和测试误差是模型关于数据集（训练集、测试集）的平均损失。

4.1 过拟合与模型选择

  当假设空间含有不同复杂度得模型时，就要选择参数个数与“真”模型相近的模型。一味追求对训练数据的预测能力，所选模型复杂度会比真模型更高。
  过拟合：模型包含参数多，对已知数据预测得很好，但对未知数据预测很差。

4.2 正则化与交叉验证

4.2.1 正则化

 模型选择的典型方法是正则化，结构风险最小化策略的实现，在经验风险上加一个正则化项（一般是复杂度的单增函数，如范数）：

minfinF1N∑i=1NL(yi,f(xi))+λJ(f)min_{f in {\mathcal{F}}}\frac{1}{N}\sum_{i=1}^{N}L(y_i,f(x_i))+\lambda J(f) minfinF​N1​i=1∑N​L(yi​,f(xi​))+λJ(f)

 具体见“ESL-监督学习概览”

4.2.2 交叉验证

  另一种常用的模型选择方法是交叉验证

  • 简单交叉验证：分成两部分
  • K折交叉验证：分成K部分
  • 留一法

  数据集的划分这个问题，书中有提到数据充足的情况下，将数据划分为三个部分，训练集，验证集和测试集。验证集和测试集有什么区别？

  在算法学习的过程中，测试集是固定的，但是验证集和训练集可能是变化的。比如K折交叉验证的情况下，非测试集分成K折之后，其中的K-1折作为训练集，1折作为验证集，这样针对每一个模型操作K次，计算平均测试误差，最后选择平均测试误差最小的模型。这个过程中用来验证模型效果的那一折数据就是验证集。交叉验证，就是这样一个使用验证集测试模型好坏的过程。他允许我们在模型选择的过程中，使用一部分数据（验证集）“偷窥”一下模型的效果。

4.3 泛化能力

  现实中采用最多的方法是通过泛化误差来评价学习方法的泛化能力

Rexp(f^)=Ep[L(Y,f(X)^)]=∫X×YL(y,f(X)^)P(x,y) dxdyR_{exp}(\hat{f})=E_p[L(Y, \hat{f(X)})]=\int_{\mathcal X\times\mathcal Y}L(y,\hat{f(X)})P(x,y)\, {\rm d}x{\rm d}y Rexp​(f^​)=Ep​[L(Y,f(X)^​)]=∫X×Y​L(y,f(X)^​)P(x,y)dxdy

  学习方法的泛化能力往往是通过研究泛化误差的概率上界进行的, 简称为泛化误差上界(generalization error bound)。定理：二分类问题的泛化误差上界

  注意泛化误差的定义，事实上，泛化误差就是所学习到的模型的期望风险
