发表于2024-10-08
|阅读量:

1. 总体主成分分析

统计分析中，数据变量之间可能存在相关性，于是需要考虑用不相关的变量来代替相关变量。

主成分分析（PCA）是一种数据降维的方法，属于多元统计分析方法，基本思想是：通过正交变换由少数线性无关的变量来代替线性相关的变量，并用来表示数据。主要用途是发现数据中的基本结构。

做法：

  • 首先对给定数据进行规范化，使得数据每一变量的均值为 0 ，方差为 1。
  • 之后对数据进行正交变换，原来由线性相关变量表示的数据，通过正交变换变成由若干个线性无关的新变量表示的数据。

新变量是可能的正交变换中变量的方差的和(信息保存)最大的，方差表示在新变量上信息的大小。将新变量依次称为第一主成分、第二主成分等。

几何意义：主成分分析等价于进行坐标轴旋转变换，将数据投影到新的坐标轴上。第一坐标轴就是第一主成分，第一主成分就是在旋转变换中选取里样本点的距离平方和最小的轴，数据在每一轴上的坐标值的平方表示对应变量的方差，以此类推。

1.1 定义

记

  • m 维随机变量 x=(x1,… ,xm)T\mathbf{x}=(x_1,\dotso,x_m)^{T}x=(x1​,…,xm​)T
  • 均值向量 μ\boldsymbol{\mu}μ

μ=E(x)==(μ1,… ,μm)T\boldsymbol{\mu}=E(\mathbf{x})==(\mu_1,\dotso,\mu_m)^{T} μ=E(x)==(μ1​,…,μm​)T

  • 协方差矩阵 Σ\SigmaΣ

Σ=cov(x,x)=E[(x−μ)(x−μ)T]\Sigma=cov(\mathbf{x},\mathbf{x})=E[(\mathbf{x}-\boldsymbol{\mu})(\mathbf{x}-\boldsymbol{\mu})^T] Σ=cov(x,x)=E[(x−μ)(x−μ)T]

考虑 mmm 维随机变量 xxx 到 mmm 维随机变量 y=(y1,… ,ym)T\mathbf{y}=(y_1,\dotso,y_m)^{T}y=(y1​,…,ym​)T的线性变换，

yi=αiTx=(α1i,… ,αmi)(x1,… ,xm)Ty_i=\alpha_i^T\mathbf{x}=(\alpha_{1i},\dotso,\alpha_{mi})(x_1,\dotso,x_m)^{T} yi​=αiT​x=(α1i​,…,αmi​)(x1​,…,xm​)T

于是有，

\begin{align*} E（y_i）&=\alpha_i^T\mu \\ var（y_i）&=\alpha_i^T\Sigma \alpha_i \\ cov(y_i,y_j)&=\alpha_i^T\Sigma \alpha_j \end{align*}

那么什么是总体主成分？给定一个如上所示的线性变换 yi=αiTxy_i=\alpha_i^T\mathbf{x}yi​=αiT​x ，如果它们满足以下条件：

  1. （标准正交基）系数向量 αiT\alpha_i^TαiT​ 是单位向量，即

    αiTαi=1,对于 i=1,2,…,m.\alpha_i^T \alpha_i = 1, \quad \text{对于 } i = 1, 2, \ldots, m. αiT​αi​=1,对于 i=1,2,…,m.

  2. 变量 yiy_iyi​ 之间互不相关，即

    cov(yi,yj)=0,(i≠j).\text{cov}(y_i, y_j) = 0, \quad (i \neq j). cov(yi​,yj​)=0,(i​=j).

  3. 变量 yiy_iyi​ 是所有线性变换中方差最大的。首先 y1y_1y1​ 是与 x1,x2,…,xmx_1, x_2, \ldots, x_mx1​,x2​,…,xm​ 中的所有线性变换中方差最大的；其次，在与 y1y_1y1​（和其他前面求得的主成分）不相关的所有变换中，y2y_2y2​ 是方差最大的。

1.2 主要性质

首先阐述定理：总体主成分与协方差矩阵特征值和特征向量的关系，以及求主成分方法。PCA 的核心：通过特征值分解来实现数据降维的关键思想。

定理 1：设 xxx 是 mmm 维随机变量，Σ\SigmaΣ 是 xxx 的协方差矩阵，Σ\SigmaΣ 的特征值分别是 λ1≥λ2≥⋯≥λm>0\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_m > 0λ1​≥λ2​≥⋯≥λm​>0，特征值对应的单位特征向量分别是 α1,α2,⋯ ,αm\alpha_1, \alpha_2, \cdots, \alpha_mα1​,α2​,⋯,αm​，则 xxx 的第 kkk 主成分是

yk=αkTx=α1kx1+α2kx2+⋯+αmkxm,k=1,2,…,my_k = \alpha_k^T x = \alpha_{1k} x_1 + \alpha_{2k} x_2 + \cdots + \alpha_{mk} x_m, \quad k = 1, 2, \ldots, m yk​=αkT​x=α1k​x1​+α2k​x2​+⋯+αmk​xm​,k=1,2,…,m

var(yk)=αkTΣαk=λk,k=1,2,…,m\text{var}(y_k) = \alpha_k^T \Sigma \alpha_k = \lambda_k, \quad k = 1, 2, \ldots, m var(yk​)=αkT​Σαk​=λk​,k=1,2,…,m

即协方差矩阵 Σ\SigmaΣ 的第 kkk 个特征值。

这个定理指出，在 PCA 中，数据的主成分是数据协方差矩阵的特征向量，这些特征向量按其对应的特征值大小排序。每个主成分的方差正好与其对应的特征值相等。

  1. y=ATx,  A 为正交矩阵\mathbf{y} = A^T \mathbf{x}, \; A \text{ 为正交矩阵} y=ATx,A 为正交矩阵

    cov(y)=diag(λ1,λ2,…,λm)\text{cov}(\mathbf{y}) = \text{diag}(\lambda_1, \lambda_2, \ldots, \lambda_m) cov(y)=diag(λ1​,λ2​,…,λm​)

    满足 λ1≥λ2≥⋯≥λm\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_mλ1​≥λ2​≥⋯≥λm​

Σαk=λkαk,k=1,2,…,m\Sigma \alpha_k = \lambda_k \alpha_k, \quad k = 1, 2, \ldots, m Σαk​=λk​αk​,k=1,2,…,m

1.3 主成分个数

以下两个定理说明了 提取 k 个主成分能够最大保留变量信息:

定理 2：对任意正整数 qqq，1≤q≤m1 \leq q \leq m1≤q≤m，考虑正交线性变换：

y=BTx\mathbf{y} = B^T \mathbf{x} y=BTx

其中 y\mathbf{y}y 是 qqq 维向量，BTB^TBT 是 q×mq \times mq×m 矩阵，令 y\mathbf{y}y 的协方差矩阵为：

Σy=BTΣB\Sigma_y = B^T \Sigma B Σy​=BTΣB

则 Σy\Sigma_yΣy​ 的迹 tr(Σy)\text{tr}(\Sigma_y)tr(Σy​) 在 B=AqB = A_qB=Aq​ 时取得最大值，其中矩阵 AqA_qAq​ 由正交矩阵 AAA 的前 qqq 列组成。

这个定理说明，通过选择合适的子矩阵 B=AqB = A_qB=Aq​，可以使线性变换后的变量 y\mathbf{y}y 的协方差矩阵的迹达到最大值。矩阵 AqA_qAq​ 是包含原始数据协方差矩阵前 qqq 个特征向量的变换矩阵。

定理 3：考虑线性变换：

y=BTx\mathbf{y} = B^T \mathbf{x} y=BTx

这里 BTB^TBT 是 p×mp \times mp×m 矩阵，ApA_pAp​ 由 Σ\SigmaΣ 的后 ppp 个主成分组成，且当 B=ApB = A_pB=Ap​ 时，Σy\Sigma_yΣy​ 的迹 tr(Σy)\text{tr}(\Sigma_y)tr(Σy​) 取得最小值。

这意味着，舍弃 AAA 的后 ppp 列，即后 ppp 个主成分时，原有变量方差的信息损失最少。

定义 1 ：第 kkk 主成分 yky_kyk​ 的方差贡献率定义为第 kkk 个方差与所有方差之和的比例：

ηk=λk∑λi\eta_k = \frac{\lambda_k}{\sum \lambda_i} ηk​=∑λi​λk​​

kkk 个主成分 y1,y2,…,yky_1, y_2, \ldots, y_ky1​,y2​,…,yk​ 的累积方差贡献率定义为 kkk 个方差之和与所有方差之和的比例：

∑i=1kηi=∑i=1kλi∑λi\sum_{i=1}^{k} \eta_i = \frac{\sum_{i=1}^{k} \lambda_i}{\sum \lambda_i} i=1∑k​ηi​=∑λi​∑i=1k​λi​​

通常取 kkk 使得累积方差贡献率达到理想的百分比，例如 70%~80% 以上。

定义 2：kkk 个主成分 y1,… ,yky_1,\dotso,y_ky1​,…,yk​ 对原有变量 xix_ixi​ 的贡献率定义 νi\nu_iνi​ 两者的相关系数的平方，：

νi=ρ2(zi,(y1,y2,…,ym))\nu_i = \rho^2(z_i, (y_1, y_2, \ldots, y_m)) νi​=ρ2(zi​,(y1​,y2​,…,ym​))

计算如下：

νi=ρ2(zi,(y1,y2,…,ym))=∑j=1pρ2(yj,xi)=∑j=1pλjαij2σii\nu_i = \rho^2(z_i, (y_1, y_2, \ldots, y_m)) = \sum_{j=1}^{p} \rho^2(y_j, x_i) = \sum_{j=1}^{p} \frac{\lambda_j \alpha_{ij}^2}{\sigma_{ii}} νi​=ρ2(zi​,(y1​,y2​,…,ym​))=j=1∑p​ρ2(yj​,xi​)=j=1∑p​σii​λj​αij2​​

这部分解释了如何通过选取合适的主成分数量来保持变换后的数据的主要信息，同时计算各个变量与主成分之间的相关性。

1.4 规范化变量的总体主成分

实际问题中，不同变量会有量纲，所以需要对各个随机变量实施规范化，使其均值为 0 ，方差为 1。所以规范化随机变量 xi⋆x_i^\starxi⋆​ 的协方差矩阵就是相关矩阵 RRR 。

这部分内容描述了规范化随机变量的总体主成分的性质，概括下就是：特征值，特征值的和，特征变量，特征变量按行求和，特征变量按列求和。

  1. 规范化随机变量主成分的协方差矩阵是 $$\Lambda^=\mathrm{diag}(\lambda_1^, \lambda_2^, \cdots, \lambda_m^)$$
    其中 λ1∗≥λ2∗≥⋯≥λm∗≥0\lambda_1^*\geq \lambda_2^*\geq \cdots \geq \lambda_m^* \geq 0λ1∗​≥λ2∗​≥⋯≥λm∗​≥0 为相关矩阵 RRR 的特征值
  2. 协方差矩阵的特征值之和为 mmm $$\sum\limits_{k=1}^m \lambda_k^*=m$$
  3. 规范化随机变量 xi⋆x_i^\starxi⋆​ 与 主成分 yk⋆y_k^\staryk⋆​ 的相关系数（因子负荷量） $$\rho(y_k^, x_i^)=\sqrt{\lambda_k^}e_{ik}^, k,i=1,2,\cdots,m$$
    其中，eik∗=(e1k∗,e2k∗,… ,emk∗)Te_{ik}^*=(e_{1k}^*,e_{2k}^*,\dotso,e_{mk}^*)^Teik∗​=(e1k∗​,e2k∗​,…,emk∗​)T 是相关矩阵 RRR 对应特征值 λk⋆\lambda_k^\starλk⋆​ 的单位特征向量
  4. 所有规范化随机变量 xi⋆x_i^\starxi⋆​ 与主成分 yk∗y_k^*yk∗​ 的相关系数的平方和等于 λk∗\lambda_k^*λk∗​ $$\sum\limits_{i=1}^m\rho^2(y_k^,x_i^)=\sum\limits_{i=1}^m\lambda_k^*e_{ik}^{2}=\lambda_k^,k=1,2,\cdots,m$$
  5. 所有规范化随机变量 xi⋆x_i^\starxi⋆​ 与主成分 yk∗y_k^*yk∗​ 的相关系数的平方和等于 1 $$\sum\limits_{k=1}^m\rho^2(y_k^,x_i^)=\sum\limits_{k=1}^m\lambda_k^*e_{ik}^{*2}=1,i=1,2,\cdots,m$$

2.3 数据矩阵的奇异值分解

对于 m×nm \times nm×n 实矩阵 AAA，假设其秩为 rrr，0<k<r0 < k < r0<k<r，矩阵 AAA 可以进行截断奇异值分解：

A≈UkΣkVkTA \approx U_k \Sigma_k V_k^T A≈Uk​Σk​VkT​

  • UkU_kUk​ 是 m×km \times km×k 矩阵
  • VkV_kVk​ 是 n×kn \times kn×k 矩阵
  • Σk\Sigma_kΣk​ 是 k×kk \times kk×k 的对角矩阵

UkU_kUk​, VkV_kVk​ 分别由 AAA 的完全奇异值分解中的矩阵 UUU, VVV 的前 kkk 列取到，Σk\Sigma_kΣk​ 由 Σ\SigmaΣ 的前 kkk 个对角线元素得到。

X′TX′=(1n−1XT)(1n−1XT)T=1n−1XXTX'^T X' = \left( \frac{1}{\sqrt{n-1}} X^T \right) \left( \frac{1}{\sqrt{n-1}} X^T \right)^T = \frac{1}{n-1} XX^T X′TX′=(n−1​1​XT)(n−1​1​XT)T=n−11​XXT

SX=XXTS_X = X X^T SX​=XXT
